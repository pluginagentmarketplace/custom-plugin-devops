# Three Pillars of Observability

## 1. Metrics
- Prometheus, Datadog
- Time-series data
- Alerting thresholds

## 2. Logs
- ELK Stack, Loki
- Structured logging
- Centralized collection

## 3. Traces
- Jaeger, Zipkin
- Distributed tracing
- Request flow visualization

## OpenTelemetry
Unified standard for all three pillars.
